# ddev.github.io

Landing page for [ddev.github.io](https://ddev.github.io/) - an index of resources published by the [DDEV GitHub organization](https://github.com/ddev).

Without this repo, opening `https://ddev.github.io/` returns a 404. This page links to all known DDEV GitHub Pages sites:

- [Add-on Registry](https://ddev.github.io/addon-registry/)
- [Coder DDEV](https://ddev.github.io/coder-ddev/)
- [Documentation (stable)](https://ddev.github.io/ddev/en/stable/) - backup mirror of [docs.ddev.com](https://docs.ddev.com)
- [Documentation (latest)](https://ddev.github.io/ddev/en/latest/)
- [Sponsorship Data](https://ddev.github.io/sponsorship-data/)
